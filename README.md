# QR Code Generator

A browser-based tool that generates QR codes from text, URLs, and other data. Built with Next.js and runs entirely in your browser.

ðŸ”— **Live Demo:** [QR Code Generator](https://toolsvana.com/tool/qr-code-generator)

## What it does

This tool creates QR codes that can be scanned by smartphones. Enter any text, URL, or data and get a downloadable QR code image instantly.

## Key features

- Generates QR codes instantly
- Supports text, URLs, contact info, WiFi credentials
- High-resolution PNG output
- Works offline
- No data sent to servers
- Customizable size

## How to use

1. Enter text or URL
2. QR code appears automatically
3. Download as PNG image

## Technical details

Built with Next.js 16, TypeScript, and Tailwind CSS. Uses the qrcode library for QR code generation.

## Running locally

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```
